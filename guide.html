Cryptocurrency Price Chart

A real-time cryptocurrency price chart application that allows you to monitor cryptocurrency prices, analyze technical indicators, and track your favorite pairs.

Features

  • Support for multiple cryptocurrency exchanges (Binance and Kraken)
  • Real-time price tracking for all USDT trading pairs
  • Multiple timeframe options (varies by exchange)
  • Technical indicators (SMA, RSI, MACD)
  • Favorites system for quick access to preferred pairs
  • Advanced filtering and search capabilities
  • Auto-refresh functionality
  • Detailed price and volume information
  • Smart trading recommendations for short-term gains

How to Use

Trading Recommendations

The recommendations feature helps you identify potential short-term trading opportunities based on technical analysis.

  • Click the "Recommendations" button next to the title to view suggested trades
  • A progress indicator will show while the system analyzes trading pairs
  • Each recommendation shows:
    • The coin pair name
    • Current market price
    • Target prices (0.5%, 1%, and 2% gains)
    • Estimated time to reach each target
  • Historical Recommendations:
    • The system automatically stores the last 10 recommendation sets
    • Access previous recommendations using the dropdown next to the Recommendations button
    • Each stored recommendation shows its timestamp
    • Delete individual historical recommendations using the Delete button in the modal
    • Deleted recommendations are permanently removed from storage
  • Recommendations are designed for short-term trades (5-60 minutes)
  • The list updates each time you open the recommendations modal
  • The system analyzes all trading pairs using these criteria:
    • Recent price momentum (last 30 minutes)
    • Volume increase (minimum 20% above average)
    • Price volatility (sufficient for target movements)
    • Trading liquidity (among top 100 pairs by volume)

Exchange Selection

Choose your preferred cryptocurrency exchange using the dropdown menu in the top-right corner.

  • Choose between Binance and Kraken using the exchange dropdown
  • Available trading pairs and intervals will update based on the selected exchange
  • Note that Kraken uses XBT instead of BTC for Bitcoin pairs
  • Each exchange has slightly different interval options:
    • Binance: 1m, 3m, 5m, 15m, 1h, 4h, 1d
    • Kraken: 1m, 5m, 15m, 1h, 4h, 1d

Symbol Selection

Use the left sidebar to browse and select trading pairs. The interface provides multiple ways to find your desired trading pair.

  • Use the left sidebar to browse available trading pairs
  • Search for specific coins using the search box at the top of the sidebar
  • Filter coins by:
    • All: Shows all available trading pairs
    • Favorites: Shows your starred pairs
    • Top 100: Shows top 100 pairs by volume
  • Click the star (★) icon next to any pair to add/remove it from favorites
  • Click any pair to load its chart in the main view

Chart Controls

The chart control panel above the main chart provides options for timeframes and technical indicators.

  1. Timeframe Selection
    • Choose from various intervals (1m to 1d)
    • Changes take effect immediately
  2. Auto Refresh
    • Toggle automatic updates
    • 30-second refresh for minute intervals
    • 60-second refresh for hour/day intervals
  3. Technical Indicators
    • SMA (20): Simple Moving Average
    • RSI (14): Relative Strength Index
    • MACD: Moving Average Convergence Divergence
    • Toggle indicators on/off using the buttons

Price Information

The price information panel displays key metrics for the selected trading pair.

  • Current price with 24h price change (green for positive, red for negative)
  • Trading volume in USDT
  • Favorite status indicator (star icon)
  • Real-time updates with price movements

Getting Started

  1. Open index.html in a web browser
  2. The chart will initialize with BTC/USDT by default
  3. Use the left sidebar to select different trading pairs
  4. Add frequently used pairs to favorites for quick access
  5. Enable auto-refresh for real-time updates
  6. Toggle technical indicators as needed

Browser Compatibility

The application is compatible with modern web browsers that support:

  • ES6+ JavaScript
  • Chart.js
  • WebSocket connections (for real-time updates)
  • Local Storage (for saving favorites)
